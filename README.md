# 🐶 PINTO AI

PINTO AIは、個人事業主・中小企業向けのAI業務支援サービスです。

## 開発目的

「シンプルで使いやすいAIアシスタント」をコンセプトに開発しています。

現在はMVP（Minimum Viable Product）を開発中です。

## 主な機能

- AIチャット
- チャット履歴（メモリ）
- タイムスタンプ表示
- レスポンシブ対応（予定）
- OpenAI API連携（予定）

## 使用技術

- Java 21
- Spring Boot 3.5.16
- Thymeleaf
- Maven
- Git / GitHub
